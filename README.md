

## 🚀 Insurance App (LWC)

A Salesforce-based Insurance Management Application built using **Lightning Web Components (LWC)** and **Apex**. This project demonstrates core Salesforce development concepts including UI components, data handling, and backend logic.

### 🔹 Features

* Manage insurance records (create, view, update)
* Interactive UI using LWC
* Apex controllers for backend processing
* Salesforce data integration

### 🛠️ Tech Stack

* Salesforce (SFDX)
* Lightning Web Components (LWC)
* Apex
* SOQL

### ⚙️ Setup

```bash
# Clone the repo
git clone https://github.com/DeMoN-7/Insurance-App-Lwc-file

# Open in VS Code with Salesforce Extensions
# Authorize org
sfdx auth:web:login

# Deploy to org
sfdx force:source:deploy -p force-app
```

### 📌 Usage

* Deploy the project to your Salesforce org
* Open App Launcher → Navigate to the Insurance App
* Start managing records
